# agent-conveyor 0.1.8

This release carries the Codex app heartbeat lifecycle hardening from the Foil
iOS dogfood loop.

## Changes

- Added `heartbeat_recommendations.teardown_policy` to no-tmux
  `create-disposable-binding --json` output.
- Made generated manager and worker heartbeat prompts explicit that idle inbox
  polls are quiet intervals, not heartbeat teardown signals.
- Documented manager/operator ownership of terminal loop closeout with
  `finish-task --require-criteria-audit` before heartbeat teardown.

## Release Proof

Before publishing, run the package verification workflow with `version=0.1.8`
and `publish=false`, then rerun it with `publish=true` after reviewing the
verified artifact and approving the `npm-production` environment.

Local release prep should include:

- `git diff --check`
- `npm test -- --runInBand src/cli/typescript-runtime.test.ts`
- `npm run build:cli`
- smoke-generated `create-disposable-binding --json` heartbeat teardown check
- `scripts/package-smoke`
- `scripts/release-check`
